You're a Ruby programmer. Sure, you get SQL, but , you're probably proud of your beautiful code. If you use ActiveRecord, you've probably had a few moments of feeling dirty. 


Blog.filter do
  with(:created_at).greater_than(1.day.ago)
  having(:posts).with(:permalink, nil)
end

to this:


Blog.find(
  :all,
  :joins => :posts,
  :conditions => ['posts.permalink IS NULL AND blogs.created_at > ?', 'blog-post', 1.day.ago)

$ sudo gem install aub-record_filter --source=http://gems.github.com
