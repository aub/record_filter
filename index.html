If you're a Ruby programmer, you're probably proud of your beautiful code. If you use ActiveRecord, you've probably had a few moments of feeling dirty. Still, you've always felt dirty about putting it in your handsome Ruby code. You may even have a sneaking suspicion that you shouldn't have to. Now you don't. If you prefer this:


Blog.filter do
  with(:created_at).greater_than(1.day.ago)
  having(:posts).with(:permalink, nil)
end

to this:


Blog.find(
  :all,
  :joins => :posts,
  :conditions => ['posts.permalink IS NULL AND blogs.created_at > ?', 'blog-post', 1.day.ago)

Install

$ sudo gem install aub-record_filter --source=http://gems.github.com
